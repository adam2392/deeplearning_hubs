# deeplearning_hubs

Singularity images to support a bare-bones minimum image for creating Singularity images for HPC development.

Supports:
- pytorch with cuda
- tensorflow with cuda
- keras

- numpy
- scipy
- scikit-learn
- opencv
- pandas
- pytest
- flake8
- tensorboard
- tensorboardx
- tqdm
- protobuf
- onnx
- spectrum
- nibabel
- mne
